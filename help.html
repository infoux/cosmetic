1. 어떤 서비스인가요?

저희 OOO 앱에서는 개인의 피부 특성에 최적화된 맞춤형화장품을 제조 판매하는 서비스를 제공합니다. 구매하신 개인용피부진단기와 간단한 설문을 통해 고객의 피부 상태를 분석하고, 분석 결과에 따른 맞춤형 레시피를 제공, 선택하신 화장품을 주문, 제조, 배송까지 one stop 서비스로 제공합니다. 20~50대 여성 3,000명의 얼굴 피부 분석 data를 수집하여, 빅데이터 분석을 진행하였고, 분석된 자료를 활용하여, 고객님의 피부를 진단하고, 맞춤형 레시피를 제공하고 있습니다.

2. 아이디, 비밀번호 분실 시 어떻게 하나요?

로그인 화면에서 [아이디 찾기]를 클릭하여 간편찾기(이메일, 전화번호 등), 휴대폰 인증 또는 아이핀(i-pin) 인증을 통해 확인이 가능합니다. 로그인 화면에 아이디 찾기 추가 요청

로그인 화면에서 [비밀번호 찾기]를 클릭하여 아이디와 가입 시 기입한 이메일 주소 또는 휴대폰 번호를 입력하여 임시비밀번호 발급이 가능합니다. 또는 아이디와 이메일 주소로 인증 후 같은 화면에서 비밀번호 변경. (선택)
